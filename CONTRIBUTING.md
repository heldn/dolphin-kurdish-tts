# Contributing to Dolphin KURDISH TTS

Thank you for your interest in improving Dolphin KURDISH TTS! We welcome contributions from the community.

## How to Contribute

1.  **Report Bugs**: If you find a bug, please open an issue describing the problem and how to reproduce it.
2.  **Suggest Features**: Have an idea for a new feature? Open an issue to discuss it.
3.  **Submit Pull Requests**:
    *   Fork the repository.
    *   Create a new branch for your feature or fix.
    *   Write clean, documented code.
    *   Submit a pull request with a clear description of your changes.

## Development Setup

1.  Clone the repo: `git clone https://github.com/heldn/dolphin-kurdish-tts.git`
2.  Install dependencies: `pip install -r requirements.txt`
3.  Run the app: `python app.py`

## Code Style

*   Use type hints.
*   Use `logging` instead of `print`.
*   Keep the UI clean and aligned with Gradio's best practices.
